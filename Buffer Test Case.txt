# Welcome to EditMD

This is a **powerful** markdown editor with real-time preview.

## Features

- Real-time preview
- Copy to clipboard
- Download as .md file
- Clean and modern interface

## Markdown Syntax Demonstration

### Text Formatting

You can format text using:
- **Bold Text**: This is **bold** text.
- *Italic Text*: This is *italic* text.
- ***Bold and Italic***: This is ***bold and italic*** text.
